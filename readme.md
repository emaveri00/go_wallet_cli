# Go Wallet CLI
Un semplice gestore di portafoglio scritto in Go per imparare:
- Pointer Receivers vs Value Receivers
- Gestione dei cicli e dello stato
- Unit Testing in Go (`testing` package)

## Come usarlo
1. Clona il repo
2. Esegui il programma: `go run main.go`
3. Esegui i test: `go test -v`
